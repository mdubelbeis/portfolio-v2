# Mason Dubelbeis - Portfolio

A modern, performance-focused personal portfolio built with React and TypeScript.

This project showcases my early software engineering work, technical stack, and approach to frontend architecture. It is intentionally structured for clarity, scalability, and maintainability.

---

## Overview

This portfolio is a single-page application designed to present projects, experience, and technical skills in a clean and accessible format.

The focus of this build was:

- Strong component organization  
- Type-safe development  
- Responsive UI design  
- Performance optimization  
- Minimal dependency footprint  

The result is a lightweight, scalable foundation that can evolve alongside my engineering career.

---

## 🛠 Tech Stack

## Architecture & Design Approach

This project follows a modular component structure to ensure reusability and clarity.

Key architectural considerations:

- Separation of UI components and page sections
- Centralized styling strategy using Tailwind utilities
- Type definitions to reduce runtime errors
- Minimal external dependencies to preserve performance
- Clear directory organization for scalability

The design philosophy prioritizes simplicity, readability, and maintainability.

## Local Development

Clone the repository:

```bash
git clone https://github.com/mdubelbeis/portfolio-v2.git
cd portfolio-v2
npm install # Install dependencies
npm run dev # Start development server
npm run build # Create production build
```
